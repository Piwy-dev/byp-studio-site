General Knowledge Bot

Inviter le bot

General Knowledge Bot est un bot Discord de culture générale. Il proposes plusieurs commandes et fonctionalité pour tester tes connaissances en culture génrale avec les membres de ton serveur. Chaque fois qu'un membre utilise une commande de test, il gagne des points si il répond correctement et perd des points dans le cas contraire. Ces points et autres options sont sauvegardés dans une base de données. Pour plus d'informations, veuillez consulter nos conditions d'utilisation et politique de confidentialité.

Multilingue

Ce bot supporte 3 langues : Français, Anglais et Néerlandais : voir comment configurer

Liste des commandes

Commandes de test

  1. /vraifaux envoie un message contenant une proposition. Une fois fait, vous pouvez cliquer sur le bouton approprié en fonction de si vous pensez que la proposition est vraie ou fausse.
  2. /drapeau envoie un message contenant un drapeau. Une fois fait, vous pouvez entrer le nom du pays qui correspond au drapeau en cliquant sur le bouton en bas du message.
  3. /capitale envoie un message contenant le nom d'un pays. Une fois fait, vous pouvez entrer le nom de la capitale de ce pays en cliquant sur le bouton en bas du message.
  4. /logo envoie un message contenant un logo. Une fois fait, vous pouvez entrer le nom de la société qui a ce logo en cliquant sur le bouton en bas du message.

Irregular verbs commands

  1. /verbs-list send a message containing the list off all verbs in function of the following options :
    1. language : the language of the verbs list (English or Dutch).
    2. showtrad : if True display the infinitive of the verb and its translation in French ; else display the infinitive, the imperfect and the past participle.
  2. /study-infinitive documentation to come ...
  3. /study-imperfect documentation to come ...
  4. /study-participle documentation to come ...

Points commands

  1. /profile shows the points of the sellected user.
  2. /leaderboard shows the 20 first members of the server with the more points.
  3. /adminxp changes the points of the selected user. This command is only available for administrators.

Configuration commands

/setlang change the bot's language to the selected language language a supported language (French, English, Dutch)
